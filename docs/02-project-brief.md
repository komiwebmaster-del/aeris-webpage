# 02. Project Brief — AERIS Business Promo Page

> 본 문서는 시안(모바일/데스크탑)을 기반으로 페이지 구조와 각 섹션의 명세를 확정한 것입니다.

---

## 1. 프로젝트 목적

AERIS의 정체성과 솔루션 가치를 **B2B 의사결정자**(설계·시공·시설관리·발주처)와 **B2C 프리미엄 고객**(주거·리모델링) 양쪽 모두에게 효과적으로 전달하는 **단일 랜딩 페이지(One-page)**.

### 핵심 KPI
- "무료 상담 신청" 폼 전환율
- 솔루션별 "통합공조 상담 신청" 클릭률
- 평균 체류 시간 / 스크롤 깊이
- 모바일 트래픽 비중 / 모바일 전환율

---

## 2. Primary Audience

| 세그먼트 | 페이지에서 원하는 것 | 우선 노출 섹션 |
|---|---|---|
| **B2B - 설계/시공** | 시스템 사양·시공 사례·도면 자료 | Solutions (SHAT) / 적용 사례 / Contact |
| **B2B - 시설관리** | 운영 데이터·유지보수·신뢰성 근거 | Solutions / 진행 프로세스 / Contact |
| **B2B - 발주처/관급** | 레퍼런스·인증·기업 신뢰도 | 적용 사례 / ARMS(인증·근거) / Contact |
| **B2C - 프리미엄 주거** | 발코니 결로·곰팡이 해결, 쾌적함 | Hero / Why AERIS / 적용 사례 (발코니) |

### 4.1 Hero
- **배경:** Deep Navy
- **Breadcrumb-style 라벨:** "발코니 · 실내 공기환경 솔루션"
- **헤드라인:** "발코니부터 시작하는 / 완전한 실내 / 공기환경 설계" (3줄, 화이트)
- **본문:** "공기의 흐름·역류·환기 부족을 한 번에. AERIS가 공간을 분석하고 시스템을 설계합니다."
- **태그:** `ENGINEERED` `CLEAN` `RELIABLE` (3개 pill, 라이트 블루 텍스트)
- **CTA 2개:** [지금 문의하기] (Primary) + [솔루션 보기 →] (Ghost)
- **우측:** 키 비주얼 이미지 (데스크탑은 우측, 모바일은 헤드라인 상단)

### 4.2 Why AERIS — 4개 문제 카드
- **Eyebrow:** "왜 AERIS인가?"
- **헤드라인:** "발코니부터 실내 곳곳에서 매일 겪는 문제들"
- **본문:** "더 이상 방치할 수 없는 문제들, AERIS가 근본적으로 해결해 드립니다."
- **카드 4개 (그리드 — 모바일 2x2, 데스크탑 1x4):**

## 5. 기술 요구사항

### 필수
- **반응형:** Mobile-first, 시안 기준 모바일·데스크탑 정확히 매칭
- **다국어:** 한국어(default) / 영어 (헤더 한/a 토글)
- **SEO:** SSG/SSR, 메타태그·OG·JSON-LD (Organization, FAQPage)
- **성능:** Lighthouse 90+ (LCP < 2.5s, CLS < 0.1)
- **접근성:** WCAG AA, 키보드 네비, focus ring 명확
- **이미지:** WebP/AVIF, `next/image` 반응형 srcset

### 권장
- 다크모드 미지원 (라이트 단일)
- 애니메이션은 절제 (스크롤 등장 시 fade-up 정도)
- Cookie/Analytics 동의 배너 (PIPA 대응)
- 부드러운 anchor 스크롤
