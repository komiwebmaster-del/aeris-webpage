# Section — Process (진행 프로세스)

> **목적:** 문의부터 사후 관리까지 5단계 진행 흐름을 시각화하여, B2B/B2C 모두에게 신뢰감과 예측 가능성 제공.

---

## 시안 참조
- 모바일·데스크탑 모두 Use Cases 바로 아래
- 좌측 큰 번호 + 제목 + 설명의 5단계 리스트

## 앵커
- `#process`

---

## 레이아웃

### 모바일·데스크탑 공통
시안에서 좌측에 라벨(`| 진행 프로세스`) + 우측에 5단계 항목이 세로 정렬:

```
| 진행 프로세스
                    ① 무료 상담 신청    온라인 폼 또는 전화 접수 시 24시간 내 담당자가 연락드립니다
                    ② 현장 진단 및 측정  공간 방문하여 공기질·습도·환기량 정밀 측정 진행
                    ③ 맞춤 설계 제안     측정 데이터 기반 최적 솔루션 설계·제안
                    ④ 설치·시공          일정 협의 후 시공, 사후 점검
                    ⑤ 사후 관리          정기 점검·교체·원격 모니터링·긴급 A/S
```

- 데스크탑: 헤더 좌측 정렬, 5단계 우측 정렬 (2컬럼 레이아웃)
- 모바일: 헤더 위, 5단계 아래 (세로 스택)
- 배경: white 또는 `--color-gray-50`
- 상하 패딩: 모바일 64px / 데스크탑 128px

### 5단계 데이터

```ts
export const processSteps = [
  {
    n: 1,
    title: '무료 상담 신청',
    description: '온라인 폼 또는 전화 접수 시 영업일 기준 24시간 내 담당자가 연락드립니다.',
  },
  {
    n: 2,
    title: '현장 진단 및 측정',
    description: '공간 방문하여 공기질·습도·환기량 정밀 측정 진행.',
  },
  {
    n: 3,
    title: '맞춤 설계 제안',
    description: '측정 데이터 기반 최적 솔루션을 설계·제안.',
  },
  {
    n: 4,
    title: '설치·시공',
    description: '일정 협의 후 시공, 사후 점검.',
  },
  {
    n: 5,
    title: '사후 관리',
    description: '정기 점검·교체·원격 모니터링·긴급 A/S.',
  },
] as const;
```

i18n: `process.steps.[1~5].title`, `process.steps.[1~5].description`

## Step 항목 시각

```
┌─────┐
│  1  │  무료 상담 신청
│ ──○ │  온라인 폼 또는 전화 접수 시...
└─────┘
```

- 번호: 작은 원형 (`size 24px`, navy 배경, white 숫자) 또는 시안에 따라 라이트 블루 배경
- 단계 사이 세로 라인 (옵션, 시안 재확인) — 진행감 표현
